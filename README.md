# MedNES
MedNES is a cycle-accurate NES emulator written in C++.
Supported mappers: NROM(0), UNROM(2)

## Usage

```
git clone https://github.com/wpmed92/MedNES.git
make
./MedNES -insert <path/to/rom>
```
